# Hostel_2026
# 🏨 Hostel_2026 Web API

A robust ASP.NET Core 8 Web API project designed to manage hostel operations such as users, students, staff, wardens, rooms, complaints, and more.

## 🚀 Features
- User Authentication (Login & Register)
- Student Management
- Staff & Warden Management
- Room & Hostel Management
- Complaint Management
- Mess Management

## 🛠️ Tech Stack
- ASP.NET Core 8 Web API  
- C#  
- SQL Server  
- ADO.NET  

## 📂 Project Structure
Hostel_2026/
│
├── Controllers/
├── Models/
├── Repository/
├── appsettings.json
├── Program.cs

## 🔑 API Endpoints

### Register
POST /api/User/Register

### Login
POST /api/User/Login

## ⚙️ Configuration

Update appsettings.json:
"ConnectionStrings": { "DefaultConnection": "Server=YOUR_SERVER;Database=Hostel_2026;Trusted_Connection=True;" }

## ▶️ Run Project
1. Clone repo  
2. Open in Visual Studio  
3. Update DB connection  
4. Run
